QGIS Blending
Features & Layers

This map represents the number of lost honey bee colonies from 2017. By using the USDA's annual Honey Bee Colonies report, released August of 2018, data was extracted, cleaned, and joined to a U.S. States shapefile in QGIS. Data from the Honey report CSV file was cleaned before import into QGIS by removing empty rows and unlabeled columns. State ID codes were added to the CSV to mirror the codes within the shapefile. Once in QGIS, the table was joined to the state attribute table. A legend and a color ramp where added to emphasize the diversity in number of colonies. Further data, such as pounds of honey produced and colony stressors, will be joined to the U. S. States shapefile in the future.

ArcMap Visitor By Location

Working with the Chinese-American Center of Chicago, a map was created using ArcMap showing the visitors by zip code. Locations were manually entered into an excel document where they were then imported into ArcMap and joined to an existing zip code layer.

ArcMap Ranking By Zip Code

This project utilized ArcMap selection tools and an SQL equation to create a list of a hundred zip codes. The list was sorted based on two requirements and three preferences determined by the creator. The objective was to find the best zip code to live and work in.
